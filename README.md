# uuidserver

An http server implementation inspired by <https://github.com/syumai/uuidserver>.

## Demo

- run `uuid.New()`
  - <https://uuid.shogo82148.com/>
- run `uuid.NewV4()`
  - <https://uuid.shogo82148.com/v4>
- run `uuid.NewV7()`
  - <https://uuid.shogo82148.com/v7>
- run `uuid.Nil()`
  - <https://uuid.shogo82148.com/nil>
- run `uuid.Max()`
  - <https://uuid.shogo82148.com/max>

## CORS support

CORS is enabled. You can retrieve UUIDs using the JavaScript `fetch` function.

- Demo: <https://shogo82148.github.io/uuidserver/>
